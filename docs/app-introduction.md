## 应用描述

飞语陪伴是一款情绪陪伴与慢社交 App。我们把熟悉的漂流瓶带到移动端：你可以把说不出口的心事写进瓶子，抛向数字海洋；也可以随机拾起一只瓶，与陌生的灵魂短暂相遇。

核心功能

海洋：投掷与捡拾漂流瓶，按心情筛选，收藏触动你的文字。

广场：浏览心情动态，点赞互动，在评论区友善表达。

邂逅：回应漂流瓶后开启匿名聊天，用文字慢慢靠近彼此。

我的：写日记、解锁成就、夜航陪伴计时，管理账号与安全设置。

为什么选飞语陪伴

慢下来：不追求即时匹配与喧闹社交，更看重文字里的真实情绪。

有边界：支持举报、拉黑与屏蔽，社区规范清晰，共建安全空间。

够私密：日记默认仅存本机；邮箱验证码登录，无需暴露真实身份。

适合这样的你

想在夜深人静时找人说说话，又不想被熟人看见。

喜欢写信式的交流，愿意用几行字承载心情。

希望有一个温柔、不喧闹的情绪出口。

重要说明

飞语陪伴不提供心理咨询或医疗诊断。若你处于心理危机，请寻求专业机构或紧急热线帮助。
